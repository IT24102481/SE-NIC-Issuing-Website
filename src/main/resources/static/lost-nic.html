LOST NIC REPORT

Report Lost National Identity Card

If you have lost your NIC, please file a police report first and then complete this form. All fields are required.

Important Information

Before filling this form, you must:

  1. File a report at your local police station about the lost NIC
  2. Obtain the official police report document
  3. Have your birth certificate ready for verification
